Tài liệu Manim

Tài liệu Manim (Viết lại)

1. Tổng quan về Manim

Manim là một công cụ mã nguồn mở mạnh mẽ để tạo hoạt hình kỹ thuật, được phát triển bởi cộng đồng Manim Community (ManimCE), dựa trên phiên bản gốc của Grant Sanderson (3Blue1Brown). Manim sử dụng Python để lập trình hoạt hình, cho phép tạo ra các hình ảnh và video chính xác, phù hợp với giáo dục, khoa học và trình bày ý tưởng phức tạp.

2. Cài đặt và kiểm tra môi trường

2.1. Cài đặt cơ bản

Yêu cầu: Python 3.7 trở lên và pip.
Sử dụng lệnh sau để cài đặt Manim:

pip install manim

2.2. Kiểm tra phiên bản

Để xác nhận cài đặt thành công, chạy lệnh:

manim --version

2.3. Cài LaTeX (Windows)

Cần thiết để hiển thị công thức toán học:

winget install MiKTeX.MiKTeX

2.4. Cài FFmpeg (Windows)

Cần thiết để xuất video:

winget install FFmpeg

2.5. Cài plugin voiceover (Tùy chọn)

Thêm giọng nói vào video:

pip install manim-voiceover

3. Cấu trúc lập trình cơ bản

3.1. Mobject: Đối tượng hiển thị

Mobject (Mathematical Object) là các đối tượng hiển thị trên màn hình, ví dụ: hình học như Circle, Square; văn bản như Text, Tex; hoặc đồ thị như Graph. Tất cả những gì bạn thấy trong video Manim đều là Mobject.

3.2. Animation: Hiệu ứng chuyển động

Animation điều khiển cách Mobject thay đổi, ví dụ: FadeIn (hiển thị mờ dần), Rotate (xoay), Transform (biến đổi). Chúng được áp dụng trong Scene thông qua phương thức self.play().

3.3. Scene: Không gian hoạt hình

Scene là lớp chính để xây dựng hoạt hình. Bạn kế thừa từ Scene và định nghĩa phương thức construct() để tạo hoạt hình.

3.4. Camera: Điều khiển góc nhìn

Camera kiểm soát góc nhìn, ví dụ: zoom, xoay, hoặc di chuyển. Các loại Camera bao gồm MovingCamera, ThreeDCamera, MultiCamera.

4.6.5. LayoutFunction

from manim import *
import numpy as np
class LayoutFunctionExample(Scene):
    def construct(self):
        vertices = [1, 2, 3, 4]
        edges = [(1, 2), (2, 3), (3, 4), (4, 1)]
        layout = {v: [np.cos(i * PI/2), np.sin(i * PI/2), 0] for i, v in enumerate(vertices)}
        graph = Graph(vertices, edges, layout=layout, labels=True)
        self.play(Create(graph))
        self.wait()
